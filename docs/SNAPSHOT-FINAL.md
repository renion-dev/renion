## Технічний стан

- **Мова:** Python 3.10+
- **База даних:** SQLite (aiosqlite)
- **LLM:** Ollama (llama3:latest)
- **Веб-фреймворк:** FastAPI + Uvicorn
- **Шаблони:** Вбудований HTML
- **Архітектура:** Clean Architecture (domain / application / infrastructure / interfaces)
- **Структура проєкту:**
  - `src/domain/` — сутності (Object, Event, Payment, Invoice)
  - `src/application/` — use cases (OpportunityHunter, Analyzer, LandingGenerator, AdvertisingManager)
  - `src/infrastructure/` — Storage, EventBus, зовнішні API
  - `src/interfaces/` — CLI, Web (FastAPI)
  - `src/templates/` — шаблони лендингів

## Висновок

AION v0.3.0 є повноцінним MVP, який закриває повний цикл:

**Збір даних → Аналіз → Гіпотеза → Інвойс → Лендинг → Реклама (заглушка) → Оплата (заглушка)**

Система працює локально, використовує реальні дані з інтернету, аналізує через LLM та генерує бізнес-гіпотези з готовими лендингами та економічним шаром.

Наступний етап — інтеграція з реальними платіжними системами та рекламними платформами для виходу в продакшн.
