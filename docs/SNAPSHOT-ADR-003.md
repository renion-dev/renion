# Snapshot: ADR-003 — Повноцінний Opportunity Hunter з LLM

**Дата:** 2026-07-21  
**Статус:** Впроваджено (частково)  

## Мета ADR-003

Створити автономного агента, який:
- Аналізує десятки джерел (Reddit, GitHub Issues, Product Hunt, Hacker News, форуми, вакансії, відгуки)
- Виявляє незадоволений попит (проблеми, про які скаржаться багато людей)
- Генерує MVP (мінімально життєздатний продукт) у вигляді концепції
- Створює лендинг для збору попиту
- Запускає рекламу (заглушка на першому етапі)
- Відстежує оплати (поки що вручну або через симуляцію)
