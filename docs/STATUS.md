## Технічний борг

| № | Проблема | Рішення |
|---|----------|---------|
| 1 | `analyzer.py` може зламатися при зміні промпту | Потрібні unit-тести для парсингу JSON |
| 2 | SQLite не підходить для багатокористувацького режиму | Потрібна міграція на PostgreSQL |
| 3 | Ollama працює тільки локально | Для продакшну потрібен хмарний LLM (OpenAI, Groq, Together AI) |
| 4 | Немає моніторингу та логування помилок у продакшні | Додати Sentry або аналог |
| 5 | Немає резервного копіювання бази | Додати автоматичний бекуп SQLite |
